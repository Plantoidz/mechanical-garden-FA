# mechanical-garden-FA
Multimodal mechanical plantoid garden repository. 

## Setup

### Requirements

You need to be running a version of pythong greater than `3.10`. 

You will also need to install `ffmpeg` and `portaudio` if on Mac, you can use [Homebrew](https://brew.sh/) for this.


### Installing

Create and activate a python virtual environment:
``` sh
python3 -m venv env
source env/bin/activate
```

Install dependencies:

``` sh
pip3 install -r requirements.txt
```
